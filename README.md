# 🎬 Netflix Content Analysis | Tableau Dashboard

## 📌 Project Overview
This project analyzes Netflix's global content catalog to identify trends in **movie production by country, content type distribution, runtime patterns, and genre popularity**.

Using Tableau, I created an interactive dashboard to explore key insights from Netflix’s dataset and answer common analytical questions about the platform’s content library.

This project demonstrates my ability to transform raw data into meaningful insights through **data visualization, exploratory analysis, and dashboard design**.

🔗 **Interactive Tableau Dashboard:**  
https://public.tableau.com/app/profile/indra.ayushjav/viz/Netflix_17676773376470/Netflix

# 🎯 Key Business Questions
The dashboard answers the following questions:

1. Which countries produce the most movies available on Netflix?
2. What is the average runtime of Netflix movies?
3. Does Netflix have more Movies or TV Shows in its catalog?
4. What are the most common movie genres on Netflix?

# 📊 Key Insights

### 🌎 Top Countries Producing Netflix Movies
The analysis shows that **the United States dominates Netflix movie production**, followed by other major contributors such as:

- India
- United Kingdom
- Canada
- Japan

These countries have well-established film industries and strong distribution partnerships with Netflix.

---

### ⏱ Average Movie Runtime
The **average movie runtime on Netflix is approximately 90–100 minutes**, which aligns with the standard length of feature films across the global entertainment industry.

---

### 🎥 Movies vs TV Shows
The Netflix catalog currently contains **significantly more Movies than TV Shows**, reflecting Netflix’s strategy of hosting a large global movie library alongside its original series.

---

### 🎭 Most Popular Genres
The most common genres on Netflix include:

- Drama
- Comedy
- Documentaries
- International Movies
- Action & Adventure

Drama and Comedy dominate due to their wide audience appeal and cross-cultural accessibility.
